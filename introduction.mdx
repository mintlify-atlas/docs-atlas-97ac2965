## What is Open Mushaf Native?

Open Mushaf Native is a modern, minimalist Quran Mushaf built for seamless reading and Tafseer exploration. It emphasizes performance, offline access, and a distraction-free experience, powered by React Native and Expo.

The app is completely **free and open source**, designed to make reading and studying the Quran accessible to everyone across all platforms.

## Available Tafseer Sources

The app provides access to multiple authentic Tafseer sources:

- **Tafseer Al-Baghawi** - Classic interpretation by Imam Al-Baghawi
- **Tafseer Ibn Kathir** - Comprehensive Tafseer by Imam Ibn Kathir
- **I'rab Al-Quran** - Grammatical analysis of Quranic text
- **Tafseer Al-Ma'ani** - Word meanings and explanations
- **Tafseer As-Sa'di** - Modern interpretation by Sheikh As-Sa'di
- **Tafseer At-Tabari** - Detailed Tafseer by Imam At-Tabari
- **Tafseer Al-Qurtubi** - Comprehensive work by Imam Al-Qurtubi
- **Tafseer Al-Muyassar** - Simplified Tafseer from King Fahd Complex
- **Asbab An-Nuzul** - Reasons for revelation by Al-Wahidi
- **At-Tahrir wa At-Tanwir** - Modern scholarly work by Ibn Ashur
- **Al-Muyassar** - Simplified interpretation by Sheikh Tantawi

## Technology Stack

Built with modern technologies for optimal performance:

- **React Native 0.81** - Cross-platform mobile framework
- **Expo SDK 54** - Managed workflow and tools
- **TypeScript** - Type-safe development
- **Jotai** - Lightweight state management
- **PWA Support** - Installable web application

  The app uses the new React Native architecture for improved performance and future compatibility.
